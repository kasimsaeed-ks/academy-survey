# WASL Academy Parent Survey

This is a survey application for collecting feedback from parents in Islamabad about their educational needs and preferences.

## How to Run Locally

1. Make sure you have Node.js installed
2. Clone this repository
3. Run `npm install` to install dependencies
4. Run `npm start` to start the development server
5. Open http://localhost:3000 in your browser

## How to Deploy to Vercel

1. Push this entire project to your GitHub repository
2. Connect your GitHub repository to Vercel
3. Vercel will automatically detect it as a React app and deploy it

## Survey Sections

- About You & Your Child
- Current After-School Habits
- Pain Points & Frustrations
- Interest & Willingness
- Preferences & Logistics
